# geegler
Mini PHP Framework

to install using composer create a composer.json in your directory

    {
           "minimum-stability": "dev",
 			   "require":{
		                 "geegler/geegler": "dev-master"
        			     },
           "repositories": [
                   {
	          "type": "vcs",
	          "url": "http://github.com/geegler/geegler"
	        }
            ]

    }
